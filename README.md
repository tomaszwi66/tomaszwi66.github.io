## How to deploy

This is a single-file GitHub Pages site.

1. Create a repository named `tomaszwi66.github.io`
2. Upload `index.html` and `images/` folder to the root
3. GitHub Pages activates automatically for `*.github.io` repositories
4. Your site will be live at **https://tomaszwi66.github.io**
